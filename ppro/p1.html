Arithmetic Table Printer in Python

Statement:
The program takes user inputs for an arithmetic operation, the type of arithmetic table (normal or reversal), and the size of the table. It then generates and prints the arithmetic table based on the inputs.
Algorithm:
1. Prompt the user to enter the value of 'a'.
2. Prompt the user to enter the starting value 'i'.
3. Prompt the user to enter the limit of the table 'n'.
4. Prompt the user to select the arithmetic operation ('1' for addition, '2' for subtraction, '3' for multiplication, '4' for division).
5. Prompt the user to select the table type ('1' for normal, '2' for reverse).
6. Based on the selected operation and table type, perform the following steps:
- If the operation is addition and the table type is normal:
- Print the addition table of 'a' from 'i' to 'n'.
- Increment 'i' by 1 in each iteration.
- If the operation is addition and the table type is reverse:
- Print the addition table of 'a' from 'n' to 'i'.
- Decrement 'n' by 1 in each iteration.
- If the operation is subtraction and the table type is normal:
- Print the subtraction table of 'a' from 'i' to 'n'.
- Increment 'i' by 1 in each iteration.
- If the operation is subtraction and the table type is reverse:
- Print the subtraction table of 'a' from 'n' to 'i'.
- Decrement 'n' by 1 in each iteration.
- If the operation is multiplication and the table type is normal:
- Print the multiplication table of 'a' from 'i' to 'n'.
- Increment 'i' by 1 in each iteration.
- If the operation is multiplication and the table type is reverse:
- Print the multiplication table of 'a' from 'n' to 'i'.
- Decrement 'n' by 1 in each iteration.
- If the operation is division and the table type is normal:
- Print the division table of 'a' from 'i' to 'n'.
- Increment 'i' by 1 in each iteration.
- If the operation is division and the table type is reverse:
- Print the division table of 'a' from 'n' to 'i'.
- Decrement 'n' by 1 in each iteration.
7. End of the program.
